# Basic DevOps CI/CD Pipeline using GitHub Actions

## 📌 Project Overview
This project demonstrates a simple DevOps pipeline using GitHub Actions.  
A Node.js application is automatically built and tested whenever code is pushed to the repository.

## 🛠️ Technologies Used
- Node.js
- GitHub
- GitHub Actions

## ⚙️ CI/CD Workflow
- Code is pushed to GitHub
- GitHub Actions triggers automatically
- Dependencies are installed
- Application setup is verified

## 🚀 Key Features
- Automated pipeline
- Continuous Integration
- Easy deployment workflow

## 🎯 Outcome
This project helped in understanding how CI/CD pipelines work in real-world DevOps practices.
